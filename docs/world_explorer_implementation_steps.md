## 设计决策

- 迷雾之外不保存真实地形。
- 服务端拥有路线和地块生成的权威状态。
- 前端只做动画、输入和增量合并。
- 第一版随机探索默认生成不重复路线。
- 第一版手动探索先支持服务端可校验的直线/折线路线。
